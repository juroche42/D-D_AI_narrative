# US-06-01 — Service OpenAI robuste

## Objectif

Centraliser toutes les interactions avec l'API OpenAI dans un service dédié (`openAIService`) avec retry automatique, timeouts, gestion d'erreurs normalisée et support du streaming SSE.

## Fonctions exportées

### `complete(options: CompletionOptions): Promise<CompletionResult>`

Complétion standard non-streaming. Inclut retry + timeout.

```typescript
const result = await complete({
  messages:    [{ role: 'user', content: 'Décris la taverne' }],
  maxTokens:   1000,
  temperature: 0.8,
});
console.log(result.content);       // texte généré
console.log(result.totalTokens);   // tokens consommés
```

### `completeStream(options: StreamingOptions): Promise<void>`

Streaming token par token — prêt pour US-06-04 (narration D&D en temps réel).

```typescript
await completeStream({
  messages: [{ role: 'system', content: systemPrompt }, { role: 'user', content: action }],
  onChunk:  (token) => res.write(`data: ${token}\n\n`),
  onDone:   () => res.end(),
  onError:  (err) => console.error(err),
});
```

### `embed(options: EmbeddingOptions): Promise<EmbeddingResult>`

Génère un vecteur de 1536 dimensions — prêt pour US-06-02 (RAG pgvector).

```typescript
const result = await embed({ text: 'Règles D&D 5e combat' });
// result.vector → number[] de 1536 dimensions
```

### `testConnection(): Promise<{ ok: boolean; model: string; latencyMs: number }>`

Teste la connexion OpenAI — utilisé par `GET /api/ai/health`.

## Politique de retry

| Erreur | Retry ? |
|--------|---------|
| 429 Rate limit | Oui (backoff exponentiel + jitter) |
| 500 / 502 / 503 | Oui |
| ECONNRESET / ETIMEDOUT | Oui |
| 400 Bad Request | Non |
| 401 Unauthorized | Non |
| 404 Not Found | Non |

**Backoff** : `min(1000 * 2^attempt, 16000) + random(0, 500)` ms

**Max retries** : configuré via `OPENAI_MAX_RETRIES` (défaut : 3)

## Endpoint healthcheck

```
GET /api/ai/health
Authorization: Session cookie requise
```

**Réponse 200 :**
```json
{
  "success": true,
  "data": {
    "openai": "connected",
    "model": "gpt-4o-mini",
    "latencyMs": 423
  }
}
```

**Réponse si clé invalide :**
```json
{
  "success": true,
  "data": {
    "openai": "unreachable",
    "model": "gpt-4o-mini",
    "latencyMs": 12
  }
}
```
