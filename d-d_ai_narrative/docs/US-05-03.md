# US-05-03 — Page détail d'une campagne

## Vue d'ensemble

Cette US implémente la page de détail d'une campagne (`/campaigns/[id]`) avec hero section immersive, infos complètes (lieu, quête, difficulté), et le bouton "Jouer cette campagne" qui crée un salon et associe la campagne en une seule action. Le `systemPrompt` est toujours absent des réponses API et du HTML rendu.

---

## Architecture

```
app/
├── api/campaigns/[id]/
│   └── route.ts               ← GET /api/campaigns/:id — public, sans auth
│
└── (campaigns)/campaigns/[id]/
    └── page.tsx               ← Server Component — SSR + metadata SEO

components/campaigns/
└── CampaignDetail.tsx         ← Client Component — hero, détails, CTA

app/(lobby)/lobby/
└── actions.ts                 ← +playCampaignAction() — créer salon + associer campagne
```

---

## Fonctionnement du bouton "Jouer cette campagne"

```
Clic "Jouer cette campagne"
  ├── Non connecté → router.push('/login?callbackUrl=/campaigns/[id]')
  └── Connecté →
       playCampaignAction(campaignId)
         ├── createRoom(userId, username)          ← salon WAITING
         ├── prisma.room.update({ campaignId })    ← association campaign → salon
         └── return { success: true, roomCode }
       router.push('/lobby/[CODE]')
```

## Règles métier

- `systemPrompt` destructuré avant `success()` dans la route — jamais dans le payload
- `systemPrompt` destructuré avant de passer les données au composant dans `page.tsx`
- Campagnes Premium → bouton ambre désactivé, logique de paiement reportée en Epic-08
- Non connecté → redirection `/login?callbackUrl=/campaigns/[id]` (pas d'erreur affichée)
- `generateMetadata` pour le SEO dynamique (title = nom campagne, description = synopsis)
- Pattern route dynamique : inline try/catch + `error(err)` (même pattern que `GET /api/rooms/:code`)

## Composant CampaignDetail

- `THEME_CONFIG` et `DIFFICULTY_CONFIG` copiés depuis `CampaignCard.tsx` (pas importés — fichier client)
- `useTransition` pour le pending state du bouton sans bloquer l'UI
- `error` state local pour afficher les erreurs de création de salon
- Colonne CTA sticky (`top-6`) pour rester visible au scroll
